
- This folder contains the datasets for use with TimeVAE

- GME_10-20.csv contains the raw stock data from the GME stock between 11/01/2010 - 11/01/2020

- GME_20-22.csv contains the raw stock data from the GME stock between 11/01/2020 - 06/10/2022 (present day)

- GME_10-20_preprocessed.npy and GME_20-22_preprocessed.npy are the preprocessed versions of GME_10-20.csv and GME_20-22.csv
