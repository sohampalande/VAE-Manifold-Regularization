- To create a virtual env to run this repository type:

conda create --name timevae python=3.7.13

conda activate timevae

- Install PyTorch

conda install python=3.7.13 pytorch torchvision matplotlib pandas -c pytorch

- Install Scikit-learn

pip install scikit-learn 

- To install all the other libraries, go to the project directory and type

pip install -r requirements.txt

- Install Seaborn

conda install seaborn

- Install open-cv

conda install opencv
